# Jake Loves Space Game

this is simple 2D agar.io inspired game where you want to eat all
circles and squares from playground.

Each time you eat something -> some sounds is generated so you compose
your own music just by moving (aka changing your cursor on the browser screen)

![game preview](/docs/game-preview-1.png)

## dev

app is builded on `create-react-app` react boilerplate

### run for local development

```bash
npm install
npm run start
```

### build

```bash
npm run build
```
